#  Interactive XR

An immersive web-based Extended Reality (XR) platform that enables users to explore interactive 3D environments directly in the browser.

---

##  Live Demo
https://interactive-xr.vercel.app/

---

##  Overview

Interactive XR is a browser-based platform built to deliver immersive 3D and XR experiences using modern web technologies. It allows users to interact with virtual environments in real time without requiring additional installations.

---

##  Features

- 🌍 Interactive 3D environments  
- 🧠 Real-time interaction  
- 🕶️ WebXR-based experience  
- ⚡ Fast and responsive UI  
- 📱 Cross-platform compatibility  
- 🎮 Immersive user experience
